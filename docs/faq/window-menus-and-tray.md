# Window, Menus & Tray

## Can I resize the window?

Yes, the window is resizable. Drag any edge or corner to adjust the size. Your preferred size is automatically saved to `settings.yaml` and restored on next launch.

The default size is 1000 × 600 pixels.

## How do I hide the menu bar?

Go to **View > Hide Elements > Menu (press Alt to show)**. The menu bar will be hidden, but you can temporarily show it by pressing the `Alt` key.

## What UI elements can I hide?

The **View > Hide Elements** submenu allows you to hide:

- **Menu** — Hide the menu bar (press Alt to show temporarily)
- **Button Icons** — Hide shortcut icons on keys
- **Button Text** — Hide shortcut labels on keys
- **Empty Buttons** — Hide keys without configured shortcuts
- **Row F** — Hide the function keys row (F1-F10)
- **Row 1/2/3** — Hide specific letter key rows

Note: These settings are disabled in Drag & Drop mode to ensure all keys remain visible for configuration.

## Are hide settings saved?

Yes, all hide element settings are saved to `settings.yaml` and persist across sessions.

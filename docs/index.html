How It Works

Three steps. Under one second.

MaxLaunchpad lives in your system tray. Summon it with a global hotkey, press a key, and you're done.

  • 1️⃣
    Summon — Alt + `Floating overlay appears instantly
  • 2️⃣
    Press a keyF for Firefox, T for Terminal...
  • 3️⃣
    DoneApp launches, overlay hides itself

Configuration

YAML profiles, themes, custom CSS

Your config is stored in human-readable YAML. Back it up, share it, version control it. Customize the look with built-in themes or your own CSS.

  • 📄
    YAML ProfilesGit-friendly, portable config
  • 🎨
    ThemesLight, dark, or follow system
  • 🔍
    SearchCtrl+F fallback when you forget
# keyboard.yaml
tabs:
  - name: Fav
    keys:
      F: /usr/bin/firefox
      T: /usr/bin/gnome-terminal
      S: /usr/bin/gnome-settings
